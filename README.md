# Vestige Index

Plataforma institucional DeFi — Sin KYC. Sin custodia. Descentralizado.

## Stack
- React 18 + Vite + Tailwind CSS v4
- TypeScript
- pnpm workspaces

## Features
- Marketplace Top 100 criptomonedas
- Indices tokenizados (DPI, MVI, DATA, PAXG, XAUt, LINK, 1INCH, OIL)
- Noticias en tiempo real via CryptoCompare
- Comunidad con comentarios y likes/dislikes
- Calculadora de interes compuesto + Conversor de divisas
- Conexion wallets: MetaMask, WalletConnect, Coinbase, Phantom
- Panel de administracion oculto
- Modo oscuro/claro
- Multidioma ES/EN/ZH
